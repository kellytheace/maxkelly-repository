Problem 1: Multiples of 3 and 5

Problem Statement

If we list all the natural numbers below 1000 thre are multiples of 3 or 5, we get 3, 5, 6 and 9. The sum of these multiples is 23.

Find the sum of all the multiples of 3 or 5 below 1000.

Theory & Approach

Naively computing 10001000 is impossible due to size, but modular arithmetic lets us keep only the last 10 digits at each step. We sum (nn mod 1010) for n = 1 to 1000.

Python Solution


result=0
for i in range (1,1000):
    if i%3 == 0 or i%5 == 0:
        result=result+i
print(result)

Final Answer

233168
