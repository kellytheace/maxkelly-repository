Problem 1: Multiples of 3 and 5

Problem Statement

If we list all the natural numbers below 10 that are multiples of 3 or 5, we get 3, 5, 6 and 9. The sum of these multiples is 23.

If we list all the natural numbers below 1000 thre are multiples of 3 or 5, we get 3, 5, 6 and 9. The sum of these multiples is 23.

Find the sum of all the multiples of 3 or 5 below 1000.

Theory & Approach

Theory & Approach

Scalable solution (Gauss + inclusion–exclusion)

The sum of the first n natural numbers is n(n+1)/2. For multiples of a number d below a limit L, there are k = ⌊(L−1)/d⌋ such multiples (d, 2d, …, kd), and their sum is d · k · (k+1) / 2.

We want multiples of 3 or 5, so we use inclusion–exclusion: sum(3) + sum(5) − sum(15), where each sum(d) is computed with the closed form above and we use L = 1000 (exclude the limit itself).

Note on the obvious loop

A simple loop that tests each i with i % 3 == 0 or i % 5 == 0 works and is easy to read, but it runs in O(L) time. The closed form above is O(1) and scales to very large limits.

Python Solution


def sumn(limit, d):
    k = (limit - 1) // d
    return d * k * (k + 1) // 2

limit = 1000
result = sumn(limit, 3) + sumn(limit, 5) - sumn(limit, 15)
print(result)
