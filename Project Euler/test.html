Self Powers

Problem Statement

Find the last ten digits of the series 11 + 22 + 33 + ... + 10001000.

Theory & Approach

Naively computing 10001000 is impossible due to size, but modular arithmetic lets us keep only the last 10 digits at each step. We sum (nn mod 1010) for n = 1 to 1000.

Python Solution


total = sum(pow(i, i, 10**10) for i in range(1, 1001))
print(str(total)[-10:])

Final Answer

9110846700
