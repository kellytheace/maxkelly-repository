Project 1: Discounted Cash Flow (DCF) Model

A Python-based tool for estimating intrinsic value using projected cash flows.

Project Overview

This tool calculates the present value of forecasted cash flows using a discount rate, built entirely in Python with Pandas and NumPy.

Sample Code


import numpy as np

cash_flows = [100, 110, 120]
rate = 0.08
dcf = sum(cf / (1 + rate) ** i for i, cf in enumerate(cash_flows, 1))
print(f"DCF Value: ${dcf:.2f}")

Technologies Used

  • Python 3
  • Pandas
  • NumPy
